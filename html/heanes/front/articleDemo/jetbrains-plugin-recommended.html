插件列表汇总

以下是介绍的一些插件信息概览，更多详细介绍在后面。

Key promoter

插件功能

此插件的主要功能是快捷键提示，重复使用鼠标完成某项功能超过多次时会提示你去使用已存在的快捷键或自定义快捷键

插件截图

在侧边会出现一个工具栏

这里记录了用鼠标进行某项操作的次数，如果是已经可以用快捷键的操作，插件就会提示是否使用快捷键；如果是没有建立快捷键操作的，当超过一定次数后，插件就会提示是否建立快捷键操作

快捷键提示

提示建立快捷键

插件使用

安装步骤参考插件安装，安装完成后其实就可以直接使用啦，遇到提示按提示操作即可。

Grep Console

插件功能

自定义设置控制台输出颜色，debug,info,warn,error等等级别，这样可以减少干扰，只看那些对开发者有用的重要信息。

插件截图

当console栏输出时将会有颜色区分

可以自定义配置关键字过滤，按过滤内容显示不同颜色。默认有default、error、warn、warn、debug、trace关键字配置

自定义配置

插件使用

安装步骤参考插件安装，安装完成后，当你的开发过程中有需要查看IDE的console内容时，就可以看到不同啦。

CodeGlance

插件功能

在编辑代码最右侧，实时显示一块代码缩略图(类似Sublime text)

插件截图

实时根据代码显示缩略图

可以自定义一些配置项，包括滚动设置，宽度，显示位置等。

自定义配置

插件使用

安装步骤参考插件安装，安装完成后，打开一个代码文件即可看到缩略图啦。

Translation

插件功能

选中文本后进行快捷翻译。

插件截图

快捷翻译

可以自定义一些配置项，包括翻译语言，翻译引擎配置等。

自定义配置

插件使用

安装步骤参考插件安装，安装完成后，选择文本或者鼠标指向文本 > 单击鼠标右键 > Translate。或者使用快捷键进行翻译。

翻译引擎支持有道和谷歌，更多配置和使用详见https://github.com/YiiGuxing/TranslationPlugin

String Manipulation

插件功能

驼峰式命名和下划线命名等转换，配合自定义快捷键更佳。一般在java开发中，由于数据库字段长用下划线连接，而字段名多为驼峰格式，这时此插件就可以派上用场了。

插件截图

快捷翻译

可以自定义转换的快捷键，使用更方便。

自定义配置

插件使用

安装步骤参考插件安装，安装完成后，选中代码即可进行转换。

GsonFormat

插件功能

把 JSON 字符串直接实例化成类。

插件截图

根据标准json串生成java类定义

一般开发过程中，若事先定义好接口然后再开始开发时，可以根据样例json快速生成类定义，简单配置一下后即可省去大量复制粘贴的体力活。

定义空类后使用GsonFormat插件

复杂的对象也可以处理得很好

插件使用

安装步骤参考插件安装，安装完成后，先新建一个空的类，然后使用快捷键⌘+N，选择GsonFormat，然后在弹出窗口里复制json串，即可快速生成。

Background Image Plus

插件功能

IDE增加背景图片，让代码编写过程更加有(sao)趣(qi)。

插件截图

显示效果

可以设置图片路径，配置自动切换时间，可设置透明度。

支持的设置项

透明度调节显示效果

存在不足

当切换背景图片时，若图片文件较大，则会引起几秒钟卡顿

插件使用

安装步骤参考插件安装，安装完成后，配置图片即可。

Mybatis plugin

插件功能

Dao层接口与sql文件之间快捷定位的工具。

  • 提供Mapper接口与配置文件中对应SQL的导航

  • 编辑XML文件时自动补全

  • 根据Mapper接口, 使用快捷键生成xml文件及SQL标签

  • ResultMap中的property支持自动补全，支持级联(属性A.属性B.属性C)

  • 快捷键生成@Param注解

  • XML中编辑SQL时, 括号自动补全

  • XML中编辑SQL时, 支持参数自动补全(基于@Param注解识别参数)

  • 自动检查Mapper XML文件中ID冲突

  • 自动检查Mapper XML文件中错误的属性值

  • 支持Find Usage

  • 支持重构从命名

  • 支持别名

  • 自动生成ResultMap属性

  • 快捷键: Option + Enter (Mac) | Alt + Enter (Windows)
